### 朗家

**HTML5与移动互联网开发方向**

本仓库为《朗家》2.0.0版本；

如想查看《朗家》1.0.0版本，请移步https://gitee.com/liu-jing222/langJia

##项目介绍：

《朗家》是软件学院    依托 3D 建模技术自主研发的一款住宅设计平台，通过React架构和threejs技术，开发出的界面美观、功能齐全且方便使用的 **3D 全景搭建住宅**设计专用网站。
